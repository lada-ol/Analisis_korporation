## 2. Назначение и бизнес-контекст

Данные содержат каталог всех курсов, доступных на платформе, с их описанием, категорией, сложностью и длительностью. Используются для обогащения событий обучения и анализа популярности курсов.

**Примечание для Группы А:** Данный файл НЕ ИСПОЛЬЗУЕТСЯ в практической работе №2. Он заменен на `09_products.csv` для анализа продуктов-аналогов подписки Pro.
